DTT Quiz Application

General Information

This is the DTT Quiz Application. It essentially allows you to add questions to a database and take quizzes in the discrete trial testing format. Below are some screenshots of the website that is actually running on my computer. This was designed and developed for my brother since he is in his Master's degree for Pschology, and he requested that I make this so he can study for his final test. This comes with a login system that allows the user to get an email auth code to log in, or the user can enter a password to log in. The user can change their settings, view the questions in the database, and take a randomized quiz.
